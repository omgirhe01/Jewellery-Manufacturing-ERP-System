⚠ Scale Not Connected — Setup Guide
RS232 Scale Connection Steps:
1. Connect the scale to the server PC via RS232 cable
2. Windows: Open Device Manager → Ports (COM & LPT) → Note the COM port number
3. Linux: ls /dev/ttyUSB* /dev/ttyS*
4. Set the following in your backend/.env file:
SCALE_SIMULATION_MODE=false
SCALE_PORT=COM3
SCALE_BAUDRATE=9600
5. Restart the server: uvicorn main:app --reload
